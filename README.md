# Multiple Linear Regression with Backward Elimination

## Project Overview

This project demonstrates Multiple Linear Regression (MLR) using Python and Scikit-learn. It predicts a dependent variable based on multiple independent variables and applies Backward Elimination using Statsmodels to identify the most significant features.

## Technologies Used

- Python
- NumPy
- Pandas
- Matplotlib
- Scikit-learn
- Statsmodels

## Dataset

The project uses an Investment dataset containing multiple independent variables and one dependent variable.

Example workflow:
- Load dataset
- Preprocess categorical variables using One-Hot Encoding
- Split data into training and testing sets
- Train Multiple Linear Regression model
- Predict results
- Apply Backward Elimination
- Evaluate model performance

## Key Concepts Covered

- Multiple Linear Regression
- Train-Test Split
- One-Hot Encoding
- Feature Selection
- Backward Elimination
- OLS Regression
- Coefficients and Intercept
- Model Evaluation

## Learning Outcomes

After completing this project, you will understand:

- How Multiple Linear Regression works
- How to handle categorical data
- How to train and evaluate regression models
- How Backward Elimination improves model performance
- How to identify statistically significant features
